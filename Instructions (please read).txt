How to run:

1. Insert the "index.html" and "app.py" files into a folder labelled "Chatbot" 

2. Ensure you have the Python Extension from Micrososoft installed in VScode

3. in the the VScode terminal type "pip install openai==0.28.0" 

4. in the VScode terminal type "pip install flask" 

5. in the VScode terminal type "pip install flask-session" 

6. insert API key in the app.py script (API key will be located in the hand in folder). 

7. In the VScode terminal type "python app.py" and click on the link produced in the terminal to open the webpage containing the chatbot
